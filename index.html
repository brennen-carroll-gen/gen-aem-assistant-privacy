Privacy Policy

Gen AEM Assistant — Chrome Extension

Overview

Gen AEM Assistant is an internal productivity tool built for the Gen Digital web production team. It helps authors identify page information, convert between AEM URL types, and perform bulk content operations across Gen Digital's Adobe Experience Manager sites.

Data Collection

This extension does not collect, transmit, or store any personal data. Specifically:

  • No analytics or telemetry data is collected
  • No data is sent to third-party servers
  • No user behavior or browsing history is tracked
  • No personally identifiable information is gathered

Data Storage

The extension uses Chrome's local storage API (chrome.storage.local) to persist user preferences and settings (such as selected environment, display options, and tool state). This data:

  • Remains entirely on the user's device
  • Is never transmitted externally
  • Can be cleared by removing the extension

Network Communication

The extension communicates only with Gen Digital's own AEM author instances (Adobe Experience Manager servers operated by Gen Digital). These requests are made using the user's existing authenticated browser session to perform content management operations the user has explicitly initiated. No credentials are collected or stored by the extension itself.

Permissions

The extension requests browser permissions (tabs, scripting, storage, cookies, downloads, bookmarks) solely to provide its content management functionality across Gen Digital's AEM environments. These permissions are used to:

  • Read the current tab's URL for page identification and URL conversion
  • Execute scripts on AEM author pages to extract page metadata
  • Store user preferences locally
  • Access cookies for authenticated requests to AEM author instances
  • Facilitate file downloads for bulk reports
  • Manage bookmark-based navigation features

Host Permissions

The extension's host permissions are limited to domains owned and operated by Gen Digital (norton.com, nortonlifelock.com, avast.com) and Adobe (adobeaemcloud.com, experience.adobe.com). The extension does not access any other websites.

Third Parties

This extension does not share data with any third parties. It does not include any third-party libraries, SDKs, or services that collect data.

Changes to This Policy

If this policy changes, the updated version will be posted at this URL. The extension's functionality is internal tooling with no data collection, so substantive changes are unlikely.

Contact

For questions about this privacy policy or the extension, contact the Gen Digital Web Production team.
